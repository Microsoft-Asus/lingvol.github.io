This video tutorial will teach you how to limit screen time on the iPhone. Screen Time is a new iOS 12 feature that allows you to track iPhone usage and app usage, then set a screen time limit with a passcode. Restrictions on the iPhone parental controls are great but Screen Time enhances this with options for setting time limits on apps, password protecting apps and even scheduling time kids can use the iPhone apps. You can set parental controls on apps like YouTube, Instagram or Snapchat.
