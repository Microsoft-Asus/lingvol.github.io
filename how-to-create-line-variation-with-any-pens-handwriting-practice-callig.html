It looks great to write with line variations. If you learn to control your hand properly, this can be achieved with any pens that you have in hand.

Pens and ink used:
- Muji fountain pen, F nib. Inked with Kaweco Palm Green
- Parker Vector fountain pen. Inked with Parker Quink, dark blue.

Paper:
Lamy ruling, 90g/m2
