Playing Minecraft with No Textures makes mining for ores a whole lot more difficult, so we’ll need to find alternate strategies to beat this challenge. So, let’s delete the Minecraft texture pack, load up a blank world, and test our skills when every block looks the same in this Skip the Tutorial Is it Possible Challenge!
