In this video, I have explained about "Downloading and Setting up Selenium WebDriver".

Selenium WebDriver is a popular testing tool for web automation. It is available in the form of a library.

Watch this video to learn about downloading and setting up selenium WebDriver in plain java and maven projects.

WebDriver drives a browser natively, as a user would, either locally or on a remote machine using the Selenium server, marks a leap forward in terms of browser automation.

Selenium WebDriver refers to both the language bindings and the implementations of the individual browser controlling code. This is commonly referred to as just WebDriver.

➳ WebDriver is a compact object-oriented API.
